This is kinda complicated but really isn't that hard at all.

 Want to have more MineKhan files ready-to-play without internet, but they aren't in the Launcher? Good news for you! The MineKhan Launcher is pre-set up for adding in those other files! Read below for instructions.

Download the HTML file off of wherever you have it or copy and paste off of Khan Academy into [ParentFolder]\MineKhanLauncherV[Vers.#]\Files\MKM\User-added\Template.html. Open up [ParentFolder]\MineKhanLauncherV[Vers.#]\Files\MKM.html in notepad. Go to line 104 of MKM.html. Rename the name that displays in the drop-down menu to the name of the mod. Also change the link refference. Save the file and run the Launcher, and it should appear on the MineKhan mod list. To add more, read the following steps. If you are getting it off of Khan Academy, save the HTML code as a .html in notepad and then copy line 104 of MKM.html. Hit enter (return if on Mac), paste, rename the title & link, and then hit save. However, if you are using another source, you can simply download the HTML file (you'll still need to add an entry in the code).
